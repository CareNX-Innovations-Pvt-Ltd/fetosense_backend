Methods

(inner) searchMother(req, res, next) → {void}

Searches for mothers based on the provided search string. - Uses regex matching for flexible name searching. - Filters based on `searchDocumentId` (BabyBeat doctor association). - Filters based on `orgDocumentId` (Fetosense organization association). - Sorts results to prioritize names that start with the search string.
